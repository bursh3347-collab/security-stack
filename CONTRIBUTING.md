# 🤝 Contributing to security-stack

This repository is part of the **GitHub Open Source Knowledge Restructuring Project**.

## How to Contribute
1. Add a new project analysis in `projects/[project-name].md` using the TEMC template
2. Update `comparison.md` and `SOURCES.md`
3. Extract universal code patterns to `code-base/`

## Quality Standards
- TEMC scores backed by data | Counter-argument required | Stars > 500 | Active 90 days | Permissive license
