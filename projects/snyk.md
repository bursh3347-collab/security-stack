## Core Value
Snyk bridges the gap between development and security. It finds vulnerabilities in your npm dependencies, Docker images, and Terraform files, then auto-generates fix PRs. Developer-friendly UX.

## Product Suite
1. **Snyk Open Source**: Dependency vulnerability scanning (SCA)
2. **Snyk Code**: SAST with AI-powered fix suggestions
3. **Snyk Container**: Container image scanning
4. **Snyk IaC**: Infrastructure as Code scanning
5. **Snyk Vulnerability Database**: Proprietary vuln intelligence

## Why It Might NOT Be Worth It
- Proprietary — cannot self-host or fork
- Free tier limited (200 tests/month)
- Expensive for teams ($52/dev/month)
- Vendor lock-in risk
- Open alternatives (Trivy + Semgrep) cover most use cases
